## Investment Simulation Platform
증권API와 크롤링으로 얻은 주가 데이터를 유저의 투자전략에 따라 시뮬레이션 해 봄으로써 투자에 도움을 주는 플랫폼입니다. 

## Tech/framework used
- Framework : Django3.0, Django-Rest-Framework
- Language : python3.8(32/64), javascript
- OS : Ubuntu 18.0.4
- DB : MongoDB Atlas
- Cloud : AWS-EC2
